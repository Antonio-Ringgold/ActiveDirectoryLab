<h1>Active Directory Home Lab/h1>



<h2>Description</h2>
Project consists of a creating an Active Directory home lab environment using Oracle Virtual Box. Installing and configuring Windows Server OS as a domain controller, adding users with Powershell and connecting a client PC to our domain.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Active Directory</b>
- <b>Server Manager</b>
- <b>DHCP</b>
- <b>Routing and Remote Access</b>

<h2>Environments Used </h2>

- <b>Windows Server OS 2019</b>
- <b>Windows 10</b> (21H2)
- <b>VirtualBox</b>
